# 简介
这是一款基于AndroidQuick快速开发库的Gank.io客户端。  
实现的主要功能有妹纸图片、每日干货、干货目录、搜索等。

![screenshot](https://github.com/ddnosh/githubusercontent/blob/master/image/android-quickgank.jpg?raw=true)

#  接口支持
- [干货集中营](http://gank.io)
